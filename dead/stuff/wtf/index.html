A Totally Serious Whitepaper, Unfortunately

WTF Is This Website?

A brief disclaimer for software engineers, web developers, framework priests, lighthouse-score monks, and anyone else brave enough to inspect the source.

Preface:

I cut my teeth messing with markdown editors, finicking with CSS classes, and arranging <div> after <div> inside paid-for customizable Tumblr themes. I also went through Khan Academy's Intro to HTML and CSS lessons until JavaScript entered the room. Once it became programming, I lost interest. Websites have always been more of a visual medium for me: part poster, part room, part interface, part weird little ritual machine.

That is the lens this site comes from. Not a rejection of programming, exactly, but an admission that I arrived here through layout, texture, customization, and visual play before I arrived through logic, systems, and architecture.

Abstract:

This website is not merely a codebase. It is a static-web art object, a browser-native sketchbook, a haunted personal operating system, and an ongoing field report from someone who learned the internet by living inside it rather than by receiving a laminated certificate from the Temple of Best Practices.

Disclaimer for Actual Web Developers

If you are a professional web developer or software engineer, welcome. Please remove your shoes, lower your expectations, and resist the urge to judge this repository as if it were applying for a job at a company with a kombucha tap.

The code may be strange. The architecture may be theatrical. The folder structure may occasionally look like it was discovered under floorboards. This is not because I think best practices are fake. I want to learn them. I respect craft. I respect clarity. I respect the people who can build systems that do not collapse when a semicolon looks at them funny.

But coding has always been difficult for me, and I am approaching it as a web-native artist and designer first. I care about the feeling, the interface, the weird little ritual of clicking through a site and realizing it has a pulse. If the thing works, it works. If it breaks, the break becomes part of the artwork until I understand it well enough to make it break better.

This site is vibecoded with intent. Do not mistake experimental disorder for absence of thought. Some of the chaos is load-bearing.

Operating Philosophy

If it works, it works. If it breaks, it is a feature of the art until proven otherwise.

The web is not only a delivery mechanism for products, dashboards, funnels, and cookie banners. It is also an art medium. A page can be a room. A hyperlink can be a punchline. An iframe can be a window, a wound, a television, or a questionable architectural decision that somehow holds the entire building together.

I see websites as art in the spirit of artists like Rafael Rozendaal, where the browser itself becomes the canvas. The site does not need to apologize for being weird. It needs to be alive enough to justify the weirdness.

Core Principles, Such As They Are

1. The Web Is a Medium

HTML, CSS, and JavaScript are not just implementation details. They are texture, timing, gesture, layout, mood, and mischief.

2. Static Is Powerful

A plain file that runs in a browser is still one of the most democratic creative tools ever made. No backend required. No permission slip. Just upload the thing and let it haunt the internet.

3. Best Practices Matter

I want cleaner patterns, better accessibility, stronger naming, and less cursed code. I also refuse to let the fear of doing it wrong stop me from making anything at all.

4. Working Is a Valid Prototype

A functional mess teaches more than a perfect diagram that never ships. The mess can be refactored. The blank page cannot.

Spiritual Lineage

Tim Berners-Lee is everybody's Daddy anyway. The web began as linked documents and somehow became a trillion-dollar slot machine with rounded buttons. This site is an attempt to remember that a webpage can still be personal, strange, handmade, and oddly sincere.

Also, yes, W3Schools is a cool site. People can be snobby about it if they want. Snobbery has never shipped a weird little page at 3:00 a.m. while the rest of the world was asleep.

A Request to Anyone Reading the Repo

Judge the site as an evolving artwork, not as a corporate front end assignment. If you see a bad pattern, assume I may be learning in public. If you see something clever, assume it was either intentional, accidental, or both, because that is how the browser gods distribute gifts.

Helpful critique is welcome. Condescension can go stand outside with Internet Explorer.

This project is not child's play. It is play, yes, but play is how humans learn systems before the systems convince them they are not allowed to touch anything.
