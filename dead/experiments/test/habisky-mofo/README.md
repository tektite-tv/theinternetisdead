# The 1000 Year Skunkass Habisky Mofo

A tiny standalone HTML page for a cursed fictional plant: **The Thousand-Year Skunkass Habisky Mofo**.

Because apparently botany needed a swamp goblin expansion pack.

## Files

- `index.html` — the full standalone webpage with inline CSS and a CSS-only plant illustration.
- `README.md` — this file, documenting the horror we have chosen to name.

## How to View

Open `index.html` in a browser.

Or run a local server from the folder:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

On Windows PowerShell, this usually works too:

```powershell
python -m http.server 8000
```

## Description

The Thousand-Year Skunkass Habisky Mofo is a legendary marsh plant said to bloom once every thousand years, releasing a smell so foul it makes raccoons reconsider their life choices.

It is not real. Tragically, neither is most common sense.

## Suggested Use

Drop the `habisky-mofo` folder into a website, archive, experiment directory, or whatever cursed garden you are cultivating.
